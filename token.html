Obtenha seu token TikTok com controle visual

Configure o app na plataforma TikTok for Developers, autorize usando o botão abaixo e o token retornado aparece automaticamente na seção "Token capturado".

Passo 1

Configure o app

Crie as credenciais (Client Key/Secret) no painel do TikTok e defina o redirect URI para esta página (token.html).

Passo 2

Autorize

Preencha o Client Key abaixo, gere uma state única e clique em "Abrir TikTok" para liberar o diálogo de autorização.

Passo 3

Copie o token

Após concluir o fluxo, o TikTok redireciona para esta página com o token em query string. Ele aparece automaticamente abaixo.

Token capturado

Código disponível assim que o TikTok retornar o redirect

OAuth

Sem token? verifique a query string para access_token ou code.

Use esta página como ponto de recepção. O TikTok retornará query params como access_token, refresh_token ou code, dependendo da configuração da sua aplicação.

  • Defina o Client Secret apenas no back-end; nunca exponha no navegador.
  • Se estiver usando o código de autorização (code), envie para seu servidor para trocar pelo token.
  • Essa página grava o state gerado manualmente; você pode usar UUID para identificar a sessão.
